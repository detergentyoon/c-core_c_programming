# etc

`pPage-num.c` 소스 코드의 이름에 따른 코드 내용
+ `_other` : 풀이에 참조한 코드, 혹은 다른 방식으로 풀이한 코드
+ `_for_exam` : 이 문제를 풀이하기 위해 필요한 정보를 담은 코드
+ `etc...` : 그 외의 이름들은 파일명 자체에서 내용을 유추할 수 있음
